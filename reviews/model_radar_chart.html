Model Profiles

Opus 4.6

Architectural Auditor

Deepest cross-file analysis. Traces implications from SKILL.md through templates to worked examples. Highest citation precision — always includes line numbers and exact quotes. Catches structural contradictions other models miss.

Best at: Structural consistency (9), Accuracy (9)

Watch for: Occasionally misses agent-facing UX gaps

Sonnet 4.6

Balanced Pragmatist

Most balanced profile across all dimensions. Strong at code correctness — caught the Java CompletableFuture<r> typo that only it found. High agreement with Opus. Fewest false positives of any model.

Best at: Code correctness + Workflow balance (8/8)

Watch for: Conservative — may underreport documentation gaps

Haiku 4.5

Breadth Scanner

Highest finding count per review (9 average). Excellent at coverage completeness — catches missing language support faster than others. Generates more UNDOCUMENTED findings. Trade-off: higher false positive rate and lower citation precision.

Best at: Coverage completeness (8), finding volume

Watch for: Noise — some findings are stylistic, not substantive

ChatGPT

Surgical Specialist

Lowest finding count but highest signal-to-noise ratio. Excels at catching code patterns that violate the playbook's own stated rules (e.g., boundary tests using exception-only assertions contradicting anti-pattern guidance). Misses systemic gaps.

Best at: Code correctness (8), anti-pattern detection

Watch for: Blind to coverage gaps and documentation issues

Gemini Pro

Thorough Generalist

Strong across coverage and structural dimensions. Best at catching format divergences (Field Reference Table column mismatch). Tends to flag broader systemic issues rather than line-level bugs. Sometimes over-counts by treating sub-items as separate findings.

Best at: Coverage completeness (8), format consistency

Watch for: Over-generalization — may inflate finding counts
