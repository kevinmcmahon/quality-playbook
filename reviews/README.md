# Council of Three Reviews

This directory contains outputs from Council of Three methodology reviews and audit results.

## Naming Convention

```
{type}-{reviewer}-{subject}-{date}.md
```

- **type**: `council-review`, `spec-audit`, `playbook-review`
- **reviewer**: `copilot`, `cursor`, `claude`, `human`
- **subject**: what was reviewed (e.g., `methodology`, `playbook-v1.3.0`, `pilot-run-001`)
- **date**: `YYYY-MM-DD`

## Examples

- `council-review-copilot-methodology-2026-03-30.md`
- `council-review-cursor-methodology-2026-03-31.md`
- `playbook-review-claude-v1.3.0-2026-04-02.md`
