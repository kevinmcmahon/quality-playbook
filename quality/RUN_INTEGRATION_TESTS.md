## Working Directory

All commands in this protocol use **relative paths from the project root.** Run everything from the directory containing SKILL.md, quality_gate.sh, and the quality/ directory.

## Safety Constraints

- DO NOT modify source code (SKILL.md, quality_gate.sh, references/)
- DO NOT delete files
- ONLY create files in the quality/results/ directory
- If something fails, record it and move on — DO NOT fix it

## Pre-Flight Check

Before running integration tests, verify:
- [ ] `SKILL.md` exists at project root
- [ ] `quality_gate.sh` exists and is executable (`chmod +x quality_gate.sh`)
- [ ] `references/` directory contains 12 .md files
- [ ] `quality/` directory has been created (Phase 2 output)
- [ ] Bash 4+ is available (`bash --version`)

## Automated Integration Tests

### Group 1: Skill Installation Verification (UC-01)

```bash
echo "=== Group 1: Skill Installation Verification ==="
PASS=0; FAIL=0
for f in SKILL.md quality_gate.sh AGENTS.md; do
    [ -s "$f" ] && PASS=$((PASS+1)) || { echo "FAIL: $f missing or empty"; FAIL=$((FAIL+1)); }
done
for f in references/*.md; do
    [ -s "$f" ] && PASS=$((PASS+1)) || { echo "FAIL: $f missing or empty"; FAIL=$((FAIL+1)); }
done
echo "Group 1: $PASS pass, $FAIL fail"
```

### Group 2: Gate Execution on Self (UC-04)

```bash
echo "=== Group 2: Gate Execution ==="
./quality_gate.sh --general . > quality/results/gate-self-check.log 2>&1
EXIT=$?
echo "quality_gate.sh exit code: $EXIT"
[ "$EXIT" -le 1 ] && echo "PASS: Gate completed without crash" || echo "FAIL: Gate crashed (exit $EXIT)"
```

### Group 3: Functional Test Suite (UC-02)

```bash
echo "=== Group 3: Functional Tests ==="
bash quality/test_functional.sh > quality/results/functional-test.log 2>&1
EXIT=$?
tail -5 quality/results/functional-test.log
echo "Exit code: $EXIT"
```

### Group 4: Reference File Completeness (UC-01, UC-02)

```bash
echo "=== Group 4: Reference Completeness ==="
PASS=0; FAIL=0
for ref in exploration_patterns.md requirements_pipeline.md review_protocols.md spec_audit.md verification.md iteration.md constitution.md defensive_patterns.md schema_mapping.md functional_tests.md requirements_refinement.md requirements_review.md; do
    [ -f "references/$ref" ] && PASS=$((PASS+1)) || { echo "FAIL: references/$ref missing"; FAIL=$((FAIL+1)); }
done
echo "Group 4: $PASS pass, $FAIL fail"
```

### Group 8: Shell Script Syntax (Infrastructure)

```bash
echo "=== Group 8: Shell Syntax ==="
bash -n quality_gate.sh && echo "PASS: quality_gate.sh syntax OK" || echo "FAIL: quality_gate.sh has syntax errors"
bash -n quality/test_functional.sh && echo "PASS: test_functional.sh syntax OK" || echo "FAIL: test_functional.sh has syntax errors"
```
