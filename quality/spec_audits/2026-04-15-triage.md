## Triage verification probes

### Probe SA-06: "five strategies" vs four (NET-NEW)

```bash
# Confirmation proof: SKILL.md line 210 says "five strategies" but only four exist
LINE210=$(sed -n '210p' SKILL.md)
if echo "$LINE210" | grep -q 'five strategies'; then
    echo "CONFIRMED: SKILL.md line 210 says 'five strategies'"
    echo "Line 210: $LINE210"
    
    # Count actual strategies in iteration.md
    STRAT_COUNT=$(grep -c '^## Strategy:' references/iteration.md)
    echo "Actual strategy count in iteration.md: $STRAT_COUNT"
    
    # Check SKILL.md's own self-contradiction
    LINE1879=$(sed -n '1879p' SKILL.md)
    echo "Line 1879: $LINE1879"
    
    if [ "$STRAT_COUNT" -eq 4 ] && echo "$LINE1879" | grep -q 'four'; then
        echo "CONFIRMED: DIVERGENT — line 210 says five, iteration.md defines four, line 1879 says four"
    fi
else
    echo "REJECTED: Line 210 does not say 'five strategies'"
fi
```
Result: CONFIRMED. Line 210 contains "one of five strategies." iteration.md defines 4 strategies. SKILL.md line 1879 says "all four strategies." This is a self-contradiction within SKILL.md and a divergence from iteration.md.
