## 1. Summary

references/verification.md line 172 hardcodes the path `.github/skills/SKILL.md` in the version stamp consistency check instruction. When the skill is installed at `.claude/skills/quality-playbook/SKILL.md`, this path does not resolve. Same class as BUG-003 (hardcoded paths in SKILL.md).

## 2. Spec reference
