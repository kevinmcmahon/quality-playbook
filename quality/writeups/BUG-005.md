# BUG-005: 7 Required Artifacts Not Checked by quality_gate.sh

> Generated by [Quality Playbook](https://github.com/andrewstellman/quality-playbook) v1.4.0 — Andrew Stellman
> Date: 2026-04-15 · Project: quality-playbook (bootstrap self-audit)

## 1. Summary

The artifact contract table (SKILL.md lines 72-101) marks 7 artifacts as "Required: Yes" that quality_gate.sh does not check for existence: CONTRACTS.md, RUN_CODE_REVIEW.md, RUN_INTEGRATION_TESTS.md, RUN_SPEC_AUDIT.md, RUN_TDD_TESTS.md, test_functional.*, and AGENTS.md. An agent could omit all of these and still pass the mechanical gate.

## 2. Spec reference

SKILL.md line 72: "If the gate checks for it, this skill must instruct its creation."

The behavioral contract: the gate must check all Required artifacts.

## 3. The code

```
quality_gate.sh:107-108
# File existence checks present for: BUGS.md, REQUIREMENTS.md, QUALITY.md,

```

## 4. Observable consequence

An agent skipping the requirements pipeline (no CONTRACTS.md), skipping protocol generation (no RUN_*.md), and producing no functional tests (no test_functional.*) will still pass quality_gate.sh. The gate provides false assurance of completeness.
