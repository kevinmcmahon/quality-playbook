## Context

Phase 7 compares v1.5.0 against v1.4.6 across five repos (one with
formal spec content, four Spec Gap). The falsifiable hypotheses from
`QPB_v1.5.0_Implementation_Plan.md:232-236`:

- **virtio** (has formal spec): bug analysis should improve in depth;
  bugs should now carry Tier 1/2 citations + dispositions. Raw bug
  count may go up or down — analysis depth is what matters.
- **chi, cobra, express, httpx** (no formal spec): bug counts should
  stay comparable to v1.4.6. Spec Gap paths should run cleanly.
  Meta-finding: "0 Tier 1/2 requirements."
- **All repos**: zero unverified Tier 1/2 citations in any artifact.

v1.5.0 also ships three deliberately-broken variation fixtures under
`bin/tests/fixtures/phase7_variations/` (malformed citation, stale
spec, empty informal_docs) — all three are exercised by
`bin/tests/test_phase7_variations.py`; see commit p7r4 for details.
