# BUG-011 — quality_gate checks EXPLORATION.md existence only, not section structure

## Requirement
REQ-013 (quality_gate.py parses EXPLORATION.md for required section titles). The gate must
parse `EXPLORATION.md` for the required exact section titles and FAIL when any are missing.
Doc source: `[SKILL.md §Phase 1 completion gate, lines 914,919]`.

## Candidate origin
Source stage: Phase 1 exploration / Pass 1 structural review.
Candidate bug ID: CB-11.

## Location
- File: `.github/skills/quality_gate/quality_gate.py`
- Lines: 310–313
- Function(s): `check_file_existence`

## Symptom
An `EXPLORATION.md` file missing `## Quality Risks` (or any other required section) passes
the quality gate. The PASS message says only "EXPLORATION.md exists" — no section-level
validation is reported. A model that produces a structurally deficient exploration (e.g.,
only an architecture section) is not caught by the gate.

## Root cause
The check at `quality_gate.py:310-313` is:
```python
if (q / "EXPLORATION.md").is_file():
    pass_("EXPLORATION.md exists")
else:
    fail_("EXPLORATION.md missing")
```
There is no content parsing. SKILL.md:914-929 mandates five required section titles. The
gate was not updated to mirror the skill's self-gate.

## Proposed fix

```diff
--- a/.github/skills/quality_gate/quality_gate.py
+++ b/.github/skills/quality_gate/quality_gate.py
@@ -310,4 +310,20 @@ def check_file_existence(repo_dir, q, strictness):
     if (q / "EXPLORATION.md").is_file():
         pass_("EXPLORATION.md exists")
+        _check_exploration_sections(q / "EXPLORATION.md", fail_)
     else:
         fail_("EXPLORATION.md missing")
+
+def _check_exploration_sections(path, fail_):
+    """Check that EXPLORATION.md contains all required section titles."""
+    required_sections = [
+        "## Open Exploration Findings",
+        "## Quality Risks",
+        "## Pattern Applicability Matrix",
+        "## Candidate Bugs for Phase 2",
+        "## Gate Self-Check",
+    ]
+    try:
+        content = path.read_text(encoding="utf-8", errors="replace")
+    except OSError as exc:
+        fail_(f"EXPLORATION.md unreadable: {exc}")
+        return
+    for section in required_sections:
+        if section not in content:
+            fail_(f"EXPLORATION.md missing required section: {section!r}")
```
