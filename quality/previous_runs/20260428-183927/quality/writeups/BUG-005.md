# Bug Writeup: BUG-005
> Generated by [Quality Playbook](https://github.com/andrewstellman/quality-playbook) v1.5.3 — Andrew Stellman
> Date: 2026-04-28 · Project: quality-playbook

## Summary
The installed-gate packaging path in `repos/setup_repos.sh` omits the `bin/citation_verifier` dependency that `quality_gate.py` expects, so installed byte-equality citation verification silently degrades to a warning-only path.

## Spec reference
- Requirement: REQ-006
- Spec basis: `quality/REQUIREMENTS.md:210-227`
- Behavioral contract quote: "Installed targets include the verifier code needed for `quality_gate.py`"

## The code
At `repos/setup_repos.sh:195-200`, setup copies only `quality_gate.py` into the target repo's installed surface. At `.github/skills/quality_gate/quality_gate.py:32-44`, the gate soft-imports `bin.citation_verifier` and explicitly falls back to `_CITATION_VERIFIER = None` when that helper is absent. The packaging code therefore installs a gate file that cannot satisfy its own verifier import path.

## Observable consequence
On an installed target, the gate can no longer perform the same byte-equality citation checks that repo-root runs enforce. The direct symptom is that `_CITATION_VERIFIER` stays missing after install, and citation-backed findings weaken from a hard mechanical check to a warn-and-skip behavior.

## The fix
Bundle the minimal `bin/` citation-verifier surface alongside the installed gate so the standalone import path resolves the same way it does at repo root.

```diff
diff --git a/repos/setup_repos.sh b/repos/setup_repos.sh
--- a/repos/setup_repos.sh
+++ b/repos/setup_repos.sh
@@ -194,7 +194,10 @@
     # Install skill files
     mkdir -p "${dst}/.github/skills/references"
+    mkdir -p "${dst}/bin"
     cp "${QPB_DIR}/SKILL.md" "${dst}/.github/skills/SKILL.md"
     cp "${QPB_DIR}/references/"* "${dst}/.github/skills/references/" 2>/dev/null || true
     cp "${QPB_DIR}/LICENSE.txt" "${dst}/.github/skills/LICENSE.txt" 2>/dev/null || true
     cp "${QPB_DIR}/.github/skills/quality_gate/quality_gate.py" "${dst}/.github/skills/quality_gate.py" 2>/dev/null || true
+    cp "${QPB_DIR}/bin/__init__.py" "${dst}/bin/__init__.py" 2>/dev/null || true
+    cp "${QPB_DIR}/bin/citation_verifier.py" "${dst}/bin/citation_verifier.py" 2>/dev/null || true
```
